# Khan小结
  - 英文
  - 入门内容，不面向职业
  - 在线学习系统目前最高水平
  - 内容和系统都制作精良
  - 全部免费
  - 强烈推荐
---
# 探索2: 林大[Web前端编程](https://www.icourse163.org/learn/BJFU-1003382003)慕课
- HTML+ CSS + JS + 案例
- 讲解细致，知识点全面
- 不足
  - 传统课堂方式，以讲为主，学生坐不住
  - 入门内容，不面向职业
- 适合入门，强烈推荐
---
# 探索3：职校课程
- 传智
- 讲解生动、内容全面，面向职业
- 问题
  - 内容太多，进度太快，适合全职
  - 我们的学生不仅学编程，还有其他核心课程（人文、社会、艺术、职业素养）
