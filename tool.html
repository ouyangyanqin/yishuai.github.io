VSCode安装与使用

名称 内容
安装
  • 官方网站下载安装
使用方法
  1. 点击扩展按钮，搜索"Chinese"，安装"中文（简体）"语言包。安装后，按提示，重启VScode软件
  2. 点击扩展按钮，搜索"HTML Preview"，安装作者是Thomas Haakon的"HTML Preview"语言包。安装后，打开一个HTML文件，先”Ctrl+V“，松开按键，然后按k，就能在网页的右边看到网页的效果
  3. 在菜单栏选择”设置“，搜索"emmet"，在结果页面的最下面，找到”Emmet: Trigger Expansion On Tab"，将它前面的方格打上勾，这样就能够通过敲Tab键，自动扩展Emmet标签命令

Atom安装与使用

名称 内容
安装
  • 官方网站下载安装
使用方法
  1. 输入HTML tag（比如li），然后按Tab健，会自动补全
  2. Github 克隆（Clone）下来代码仓库后，用Atom打开，鼠标移到右边缘，会出来左箭头。点击Github标签页，按提示，点击链接，会自动打开浏览器，申请Github授权，然后将网页上的授权码复制到Atom下面的授权码栏内，确认，即完成了Github登录
  3. 文件修改后，点击右侧的Git标签页，点击右上角Stage All，然后输入Commit Message，点击Commit to master，然后点击最下面的Push按钮，就能将最新的修改上传到Github上
  4. 如果Github上的文件被其它方式改过，那么需要将这些改动同步到本地来。点击右下角的Fetch按钮，即可
  5. 在设置里，把 Open（打开）Empty（空的）Editor（编辑器）On（在）Start（启动）前面的“勾”，去掉。这样，就不会每次打开，都出来“空”的内容了。
  6. 在设置里，把 Restore Previous Windows（恢复先前的窗口）On（在）Start（启动），选为always，这样，每次打开一个目录（项目），就会恢复以前的窗口。

Sublime Text安装与使用

名称 内容
安装 官方网站下载安装
常见命令
  1. Ctrl + D：选择当前光标所在的词并高亮该词所有出现的位置，再次Ctrl + D选择该词出现的下一个位置，在多重选词的过程中，使用Ctrl + K进行跳过，使用Ctrl + U进行回退，使用Esc退出多重编辑
  2. Ctrl + J：把当前选中区域合并为一行
  3. Ctrl + F2：添加/移除书签
  4. F2：跳到下一个书签
  5. Ctrl + Shift + L: 对一片区域的所有行进行同时编辑
  6. Ctrl + Cmd + G: 全选选中的文本

B站投稿和视频嵌入方法

名称 内容
介绍
  1. 登录B站后，访问 B站创作中心 - 视频投稿投稿页面，上传视频。
  2. 上传视频时，不要怕麻烦，要详细给出视频的各种标签、介绍哦，这样B站才会知道如何把你的视频推荐给想寻找这些视频的人
  3. 记得在视频的介绍中给出你的Github网站链接哦，这样就会很多人来访问你的网站！
  4. 耐心等待审核通过（一小时左右）
  5. 审核通过后，参考 用iframe/embed在网页上嵌入视频/音乐组件 文中介绍方法，获得你的视频的iframe代码，粘贴到我们的Github网站代码中（可以参考下面的示例）
  6. 把文件上传到Github上，然后用浏览器访问你的网站，就可以看到你的B站视频喽！（如果是本机上打开网页，可能看不到。没关系。上传到网站上，就可以看到了。
